===============================================================================
                            IMPORTANT NOTES
===============================================================================

1. SECRET_KEY:
   - Generate a random 32-40 character string
   - Use letters, numbers, and special characters
   - PowerShell command to generate:
     -join ((65..90) + (97..122) + (48..57) | Get-Random -Count 40 | % {[char]$_})
   - Or use: https://randomkeygen.com/

2. GEMINI_API_KEY:
   - Get from: https://makersuite.google.com/app/apikey
   - Create 5-10 Google accounts to get 5-10 keys for better performance
   - Each key = 60 requests/minute
   - 10 keys = 600 requests/minute!

3. DATABASE_URL:
   - If you added PostgreSQL in Railway, use: ${{Postgres.DATABASE_URL}}
   - If using external database, use full connection string:
     postgresql+asyncpg://user:password@host:5432/database

4. REDIS_URL:
   - If you added Redis in Railway, use: ${{Redis.REDIS_URL}}
   - If using external Redis, use: redis://user:password@host:6379

5. CORS_ORIGINS:
   - Add this AFTER you deploy frontend
   - Replace with your actual frontend URL
   - Multiple origins: https://site1.com,https://site2.com

6. SMTP Settings:
   - Only needed if you want email notifications
   - For Gmail, use "App Password" not regular password
   - Get app password: https://myaccount.google.com/apppasswords
